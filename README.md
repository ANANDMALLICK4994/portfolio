# Anand Mallick - Personal Portfolio

Welcome to the repository for my personal portfolio website! This project is a fully responsive, modern web application designed to showcase my skills, projects, and professional background.

## 🚀 Features

- **Modern & Responsive UI**: Built with a mobile-first approach ensuring a seamless experience across all devices.
- **Dynamic Typing Effect**: A sleek, animated text effect in the hero section displaying various roles (Web Developer, UI/UX Enthusiast, IoT Enthusiast, PCB Designer, AI Enthusiast).
- **Scroll Animations**: Smooth reveal animations as you scroll down the page, providing an engaging user experience.
- **Glassmorphism Design**: Stylish "glass" panel effects on skill cards and project sections.
- **Sticky Navigation**: A navigation bar that stays accessible as you scroll, complete with a mobile-friendly hamburger menu.
- **Interactive Contact Form**: A ready-to-use contact section for inquiries and messages.

## 🛠️ Technologies Used

- **HTML5**: Semantic structure and accessibility.
- **CSS3**: Custom styling, CSS variables for theming, Flexbox/Grid layouts, and animations. No external UI frameworks were used.
- **JavaScript (Vanilla)**: DOM manipulation, event handling, dynamic typing effect, and scroll reveal logic.
- **FontAwesome & Devicon**: For vector icons and technology logos.
- **Google Fonts**: Utilizing the 'Outfit' font family for modern typography.

## 📂 Project Structure

- `portfolio.html`: The main HTML document containing the structure of the website.
- `style.css`: The stylesheet defining the visual presentation, including colors, typography, and responsive media queries.
- `script.js`: The JavaScript file powering the interactive elements like the typing effect, mobile menu, and scroll reveals.
- `anand1.jpeg`: Profile image used in the hero section.

## 💻 Getting Started

To view the project locally:

1. Clone this repository or download the source code.
2. Open the `portfolio.html` file in your preferred web browser. No server setup or build process is required!
